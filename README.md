## WHAT IS THIS?

This feature is for doing auto casing identified by prefix.

## Where the file can support this tool?

The tool supports:

* md file
* json file

## How to use it?

```
./auto-casing.sh --prefix="fill your prefix" --path="fill your md/json file" 
```

example:

```
./auto-casing.sh --prefix="#" --path=sample.md
```
